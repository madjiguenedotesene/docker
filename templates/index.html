Comment utiliser l'outil ?

📁

Charger

Comment Procéder ?

Cliquez sur "Commencer l'analyse" puis sur l'icône de téléchargement. Sélectionnez vos fichiers (CSV, XLSX) à importer.

Pourquoi cette étape ?

C'est le point de départ. Une fois chargés, l'outil analyse automatiquement la structure des données et leur type.

📈

Prévisualiser

Comment Procéder ?

Allez à la page Visualisation. Choisissez les colonnes que vous voulez comparer (ex: Température vs Humidité) pour générer des graphiques.

Concepts Clés

Les Couleurs et Catégories sont utilisées pour segmenter vos données, par exemple, afficher les points de données par ville ou par type d'événement.

🔮

Prédire

Comment Procéder ?

Allez à la page Modélisation. Sélectionnez une variable cible (ex: "Température") et lancez le modèle pour obtenir une prédiction.

Concepts Clés

Le Coefficient R² est le score à surveiller. Un R² de 1 (100%) signifie une prédiction parfaite.

La Corrélation mesure la relation entre deux variables (+1 = augmentent ensemble, -1 = inverse).
