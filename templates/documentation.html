Comment fonctionne le logiciel ?

Ce guide vulgarisé explique étape par étape comment vos données brutes (fichiers ou capteurs) sont transformées en graphiques et en prédictions intelligentes.

1. L'Import des Données

L'application permet d'importer des données de deux manières :

  • Fichier Local : Vous donnez un fichier Excel ou CSV, le logiciel le lit automatiquement.
  • API Eurosmart : Vous entrez un code (ex: Paris_01). Le logiciel se connecte de manière sécurisée aux serveurs de PolluGuard et télécharge les données.
En coulisses : Le logiciel détecte automatiquement la ville grâce au GPS du capteur (via OpenStreetMap) et stocke les données dans une mémoire ultra-rapide appelée Parquet.

2. Le Nettoyage Magique

Les données des capteurs sont souvent imparfaites. Le logiciel fait le ménage sans que vous n'ayez rien à faire :

  • Renommage : Les noms bizarres comme _HR deviennent "Humidité".
  • Temps : Les dates sont converties dans un format standard lisible par l'ordinateur.
  • Réparation : S'il manque une donnée (trou), le logiciel la remplace par la médiane (valeur moyenne) pour ne pas casser les graphiques.
  • Calculs : Le logiciel peut calculer de nouvelles variables scientifiques, comme le Point de Rosée.

3. La Visualisation Interactive

Vos données deviennent de véritables graphiques manipulables (zoom, survol) grâce à la technologie Plotly.

  • Courbes : Pour voir l'évolution dans le temps.
  • Secteurs & Histogrammes : Pour voir la répartition globale.
  • Nuages de points : Pour voir si deux polluants sont liés.
Catégories intelligentes : Si vous analysez le CO2, l'application ne se contente pas d'afficher le chiffre. Elle colorie le graphique en "Vert" (Confort < 800ppm) ou "Rouge" (Non-Confort).

4. Intelligence Artificielle (Prédiction)

Le logiciel n'analyse pas seulement le passé, il tente de deviner le futur (sur les prochaines 24h).

  • Machine Learning : Il utilise des modèles mathématiques avancés (Régression Linéaire, Forêts Aléatoires).
  • Corrélations : L'IA regarde quelles variables évoluent ensemble pour mieux comprendre le phénomène.
  • Lags (Mémoire) : L'IA apprend en regardant ce qu'il s'est passé à l'heure précédente pour prédire l'heure suivante.

5. Comparaison Météo

Comment savoir si le capteur de l'école est fiable ?

L'application interroge une base de données mondiale (Meteostat) pour récupérer les relevés officiels de la station météorologique la plus proche de votre ville.

Elle superpose ensuite la courbe de votre capteur avec la courbe de Météo France pour vérifier que les mesures (Température, Humidité) sont exactes.

6. Interprétation Pédagogique

C'est la fonction la plus utile pour les élèves : un moteur d'explication automatique des graphiques.

  • Détection des Pics : Le logiciel trouve le point le plus haut et le plus bas.
  • Contexte temporel : Il regarde si ce pic s'est produit la nuit, le jour, le matin ou le week-end.
  • Conclusion écrite : À partir d'une base de connaissances (ex: "Le CO2 augmente s'il y a des élèves dans la classe"), le logiciel rédige une phrase expliquant pourquoi la courbe a monté ou baissé.
